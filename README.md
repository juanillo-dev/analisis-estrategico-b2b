# Caso de Prospección B2B - Optimización de Flotas en FMCG

## Objetivo del proyecto

Este proyecto simula un caso real de prospección B2B para una solución SaaS de optimización de flotas dentro del sector FMCG.

El objetivo es estructurar un proceso completo que incluya:

Análisis sectorial  
Construcción de hipótesis comercial  
Segmentación de cuentas  
Priorización estratégica  
Definición de perfil de cliente ideal  
Planteamiento de estrategia de contacto  

No se trata de un ejercicio teórico, sino de una simulación ejecutable con lógica comercial real.

## Contexto

El sector FMCG opera con alta rotación de producto, márgenes ajustados y fuerte presión en tiempos de entrega.

Un cambio estratégico dentro del ecosistema genera impacto operativo en la cadena de distribución, especialmente en distribuidoras directas con flota propia.

## Enfoque aplicado

El proyecto se desarrolla en cuatro fases:

1. Investigación  
   Análisis del sector y definición de hipótesis de oportunidad.

2. Key Accounts  
   Mapeo estructurado de cuentas, segmentación y modelo de priorización basado en tamaño, alcance e impacto operativo.

3. Sales Process  
   Definición del perfil de cliente ideal y estrategia de entrada comercial.

4. Conclusión  
   Validación de hipótesis y planteamiento de métricas para entorno real.

## Modelo de priorización

Las cuentas se clasifican en:

A – Alta prioridad  
B – Prioridad media  
C – Prioridad baja  
N/D – Empresa ancla (referencia estratégica)

La priorización combina tamaño de flota, alcance geográfico, vinculación y exposición al cambio estratégico.

## Finalidad

El propósito del proyecto es demostrar capacidad de:

Análisis estructurado  
Pensamiento estratégico comercial  
Segmentación lógica  
Planificación de prospección B2B  

El siguiente paso en un entorno real sería validar la hipótesis mediante conversaciones con las cuentas priorizadas.
